# Modern Links Page

A premium **Linktree-inspired personal links page** built with **HTML + CSS in a single file**.
Designed with a completely original UI, modern aesthetics, smooth animations, and responsive layouts.

## ✨ Features

* 🎨 Unique world-class UI (not copied from Linktree)
* 📱 Fully responsive design
* 🌙 Modern dark futuristic theme
* ✨ Glassmorphism effects
* 🚀 Smooth hover animations & transitions
* 💫 Animated glowing background elements
* 👤 Profile hero section
* 🔗 Social media links hub
* 📂 Projects showcase section
* 📄 Resume / CTA buttons
* ⚡ Pure HTML + CSS (single file)

## 🛠 Tech Stack

* HTML5
* CSS3
* Responsive Design
* CSS Animations
* Glassmorphism UI
